# Web App Overview

The Air Boss web app provides full access to all features from your browser.

## Access

Visit [app.air-boss.com](https://app.air-boss.com) to access the web app.

## Features

The web app includes all mobile features plus:

- **Larger screen** for easier scheduling and data entry
- **Keyboard shortcuts** for power users
- **Better data tables** for viewing flight logs and reports
- **Admin dashboard** for club management

## Dashboard

Your dashboard shows:
- Flight statistics and trends
- Upcoming reservations
- Recent club activity
- Quick access to common actions

## For Club Admins

The web app is especially useful for admins:

- **Member Management** — Approve members, verify documents, manage roles
- **Aircraft Management** — Add/edit aircraft, track maintenance
- **Reports** — View flight statistics, billing reports
- **Club Settings** — Configure club preferences and requirements

## Browser Support

Air Boss works best on:
- Chrome (recommended)
- Firefox
- Safari
- Edge

## Mobile Responsive

The web app is fully responsive and works on tablets and phones, though we recommend the native mobile apps for the best experience on smaller screens.
